# Student Performance Prediction using Machine Learning

## Overview
This project applies machine learning techniques to predict student academic performance based on academic, behavioral, and demographic factors.

The system analyzes student-related features and classifies students into high-performing or at-risk categories using multiple machine learning algorithms.

## Objectives
- Predict student academic performance
- Compare multiple machine learning models
- Apply clustering and dimensionality reduction techniques
- Evaluate model performance using different metrics
- Discuss ethical implications of educational prediction systems

## Dataset
The dataset contains information about:
- Attendance rate
- Study hours
- Previous grades
- Extracurricular activities
- Parental support
- Online classes participation
- Gender and demographic attributes

## Machine Learning Techniques Used

### Supervised Learning
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- Neural Network (MLPClassifier)

### Unsupervised Learning
- K-Means Clustering
- Principal Component Analysis (PCA)

## Data Preprocessing
- Handling missing values
- Feature engineering
- One-hot encoding
- Feature scaling
- Exploratory Data Analysis (EDA)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC
- Confusion Matrix

## Technologies & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Key Findings
- Logistic Regression achieved the best overall performance.
- Study hours, attendance, and previous grades were the most important predictive features.
- PCA and clustering helped visualize hidden patterns in the dataset.

## Ethical Considerations
The project discusses:
- Bias and fairness in educational prediction
- Privacy concerns
- Risks of incorrect predictions
- Importance of human oversight
